# Automated Trading Bot with RSI & Telegram Alerts

## Description
This Python bot monitors stocks and cryptocurrencies, calculates RSI, and sends real‑time alerts to Telegram. It can simulate automatic buy/sell orders and even generate YouTube video scripts.

## Requirements
- Python 3.8+
- Telegram account
- Internet connection

## Installation
1. Download and unzip the package.
2. Install dependencies:
   pip install -r requirements.txt
3. Create .env file (see below).

## Configuration
Create .env with your Telegram credentials:
TELEGRAM_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
- Get token: [@BotFather](https://t.me/BotFather)
- Get chat ID: [@userinfobot](https://t.me/userinfobot)

## Usage
Run:
python cartera_ia_ampliada.py

## Customization
Edit WATCHLIST and RULES inside the script.
